# PyleMC's Packy

**Packy** is a Minecraft Resource Pack Tool for Bedrock Edition, only Desktop for now.

## How to use?

Latest release can be found [here](https://github.com/PyleMC/Packy/releases)

### Running

After downloading the latest version, or whichever version you prefer, simply run it.

### Running & building from source

> **Requirements**
> Golang 1.25+
> ...

You can run it with `go run .` and compile it with `go build .`.
